# Projeto Front-End: Loja de Roupas e Pizzaria

Este projeto consiste em **duas páginas web** desenvolvidas em **HTML semântico** e **CSS estilizado** como parte de uma atividade de aprendizado de front-end. O objetivo é criar páginas **interativas, responsivas e visualmente organizadas** sem o uso de frameworks.

---

## Estrutura do Projeto

| Arquivo / Pasta           | Descrição                       |
|---------------------------|--------------------------------|
| `loja_wireframe.html`     | Página HTML da loja de roupas  |
| `loja_style.css`          | Estilos CSS da loja de roupas  |
| `pizzaria_wireframe.html` | Página HTML da pizzaria        |
| `pizzaria_style.css`      | Estilos CSS da pizzaria        |
| `imagens/`                | Contém todas as imagens usadas |

> Todas as imagens devem estar dentro da pasta `imagens/` para que as páginas funcionem corretamente.

---

## Tecnologias Utilizadas

- **HTML5 semântico**: Estrutura clara e acessível do conteúdo  
- **CSS3**: Flexbox para layout responsivo, hover, sombras e animações  
- **Responsividade**: Layout adaptável a diferentes tamanhos de tela

---

## Funcionalidades

### Loja de Roupas
- Layout moderno e responsivo usando Flexbox  
- Cards de produtos contendo:
  - Imagem  
  - Nome do produto  
  - Descrição  
  - Botão de compra com efeito hover  
- Cabeçalho com navegação clara  
- Rodapé estilizado com cores neutras

### Pizzaria
- Layout de cards de pizzas responsivo  
- Cada card contém:
  - Nome da pizza  
  - Imagem  
  - Descrição  
  - Botão de compra com efeito hover  
- Cabeçalho e rodapé coloridos e com contraste  
- Efeitos visuais suaves nos cards e botões

---

## Observações
- Verifique se os arquivos CSS estão vinculados corretamente nos HTML  
- Certifique-se de que todas as imagens estão na pasta `imagens/`  
- Projeto **front-end apenas**, sem backend
